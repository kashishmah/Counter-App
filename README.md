# Counter App

A simple and interactive counter application built using HTML, CSS, and JavaScript.

## Features
- Increment, Decrement, Reset
- Prevents negative values
- Keyboard controls
- Dark mode
- Smooth animations

## Keyboard Shortcuts
- Arrow Up → Increment
- Arrow Down → Decrement
- R → Reset

## Tech Stack
- HTML
- CSS
- JavaScript
- Bootstrap
